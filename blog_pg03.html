Develope Note 3: New Menu Daltube for Dalsaram.com

Published on Feb 3, 2025 by GH Jo tag: PHP, SQL, DB, Javascript, HTML, CSS, UI

Dalsaram.com Daltube

Daltube 달튜브 PC
Daltube 달튜브 Mobile
Program Language: PHP, mySQL, css
work time: 10 days (2025/01)
DalTube — A Customizable YouTube Channel Page

Features:


Custom-built YouTube-like page for organizing video content
Tag system for filtering videos by category
Menu-based navigation for grouping related videos
Separate PC and mobile versions (responsive integration planned in the future)
Built with PHP and MySQL, DB managed via phpMyAdmin

Development Time: 10 days, solo project

Project Background:


This project was developed upon a request from the company to create a video content hub within their own website, instead of using YouTube. The goal was to allow centralized video management and an intuitive viewing experience for users, tailored to the company’s needs.

기능 소개:


유튜브 채널을 자유롭게 구성할 수 있는 전용 페이지 제작
영상에 태그를 추가하여 태그별 필터링 가능
메뉴 기능으로 콘텐츠를 주제별로 분류
PC버전과 모바일버전 각각 별도로 제작, 어드민페이지 제작 (현 구조상 반응형 통합은 추후 예정)
PHP와 MySQL 기반, phpMyAdmin으로 DB 설계 및 관리

전체 개발 기간: 10일, 1인 개발

프로젝트 배경:


회사에서 유튜브 플랫폼을 이용하지 않고, 자체 사이트에서 영상 콘텐츠를 직접 관리하고 보여줄 수 있는 시스템을 요청하였습니다. 이에 따라 사용자 편의성과 관리 효율성을 높일 수 있는 맞춤형 유튜브 채널 페이지를 개발하게 되었습니다.
